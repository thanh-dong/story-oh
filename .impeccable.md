## Design Context

### Users
Parents and teachers discover and select stories; children aged 4-12 read them independently. The interface must work for both: clear enough for a child to navigate, trustworthy enough for an adult to curate. Primary use context is tablet/phone at home or in classrooms.

### Brand Personality
**Warm, inviting, storybook.** Like opening a beloved picture book — cozy, illustrated, nostalgic. Three words: **warm, gentle, wonder.**

The emotional goal is to make reading feel like an adventure, not a chore. Parents should feel "this is safe and enriching." Kids should feel "I want to keep reading."

### Aesthetic Direction
- **Tone**: Warm storybook — soft textures, rounded forms, earthy-warm palette with pops of color
- **Theme**: Light mode primary with polished dark mode
- **References**: Picture book illustrations, cozy reading nooks, hand-drawn quality
- **Anti-references**: Neon/gaming aesthetics, corporate SaaS dashboards, cluttered edu-tech
- **Typography**: Nunito is a good fit — friendly and rounded. Keep it.
- **Color**: Current purple primary works but should feel warmer. Kid accent colors (yellow, orange, pink, green) are good — use them more intentionally.

### Design Principles
1. **Storybook warmth** — Every screen should feel like a page in a picture book, not a software interface
2. **Effortless for small hands** — Large touch targets, clear visual hierarchy, minimal cognitive load
3. **Delight through discovery** — Reward curiosity with gentle animations, playful details, and surprise
4. **Trust for grown-ups** — Clean information architecture, readable text, professional quality
5. **Responsive by nature** — Tablet-first thinking, graceful adaptation from phone to desktop
